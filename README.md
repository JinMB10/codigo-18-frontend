# Comando de GIT

## Comando para iniciar git en una carpeta

```bash
git init
```
## Comando para ver el estado de mis archivos

```
git status
```

## Comando para agregar mis archivos al stage de git

```bash
git add.
```

## Comndo para crear una versión de nuestro código junto a un comentario.

Algunos prefijos 

feat: feature (Una nueva funcionalidad)
fix : (Reparar algo)
chore : (Cosas repetitivas)
styles : (Modifico la apariencia de la web)

```bash
git commit -m "feat: Se creò el reto-01 para el portafolio"
```
